使用前请先阅读




已有功能：
1.自选或随机角色发推，用户发推（点击右下角的蓝色圆形图标）
2.npc随机回复，角色随机回复（可在设置里自定义是否回复）
3.营销号功能（字面意思，可以选择角色生成爆料）
4.信件和日记功能（根据推文和聊天记录生成）
5.小说功能，也就是”我们的故事“这一栏（输入大纲选择角色作为出场人物后可以自动生成小说，支持导出txt，右键点击小说卡片即可，）
6.论坛体功能，也在“我们的故事”里
7.匿名论坛功能（该功能里全是阴暗面发言，接受不了的可以不用）
8.标签功能（其实这个应该放第三条但是我才想起来而且懒得改了就放这里吧。双击标签可以修改，进入标签后可以查看仅带有此标签的帖子）
9.聊天功能（点击名字切换角色，点击头像可导入或导出聊天记录，查看聊天总结，聊天总结每10条总结一轮，最多保存20轮，可自主修改，双击群头像可拉新成员入群）


注：
所有网页端右键点击功能在安卓端都是长按可触发。
有不喜欢的功能可以不用但是不要因此过来骂我谢谢。
